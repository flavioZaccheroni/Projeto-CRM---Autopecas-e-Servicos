# Fase 9 - Checklist

## Entregas implementadas

- Model `Auditoria`.
- Model `Configuracao`.
- Service de auditoria.
- Service de configuracoes.
- Service de backup e restauracao.
- Controller de seguranca.
- Tela **Seguranca e Backup** no menu Sistema.
- Criacao de backup manual do banco SQLite.
- Listagem de backups disponiveis.
- Restauracao de backup validando arquivo `.db`.
- Registro de auditoria para backup, restauracao e configuracoes.
- Logger ativado na entrada do app.
- Parametros configuraveis por chave/valor.
- Testes automatizados para auditoria, configuracoes, backup e restore.

## Pendencias naturais para refinamento

- Agendamento de backup automatico.
- Controle de permissoes gerenciais por perfil.
- Politica de retencao de backups.
- Tela dedicada de usuarios/permissoes completa.
